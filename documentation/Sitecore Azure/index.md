# Sitecore Azure

## Official Sitecore Documentation and Resources

The following documents relate to the **Sitecore Azure module**. This module relies on a number of Microsoft Azure Services (and therefore utilizies Microsoft Azure in a PaaS capacity). Make sure that you are aware of the **size limitations** (especially of the Azure SQL Database service) if you decide to go down this route.

- [Sitecore Azure Compatibility Table](https://kb.sitecore.net/articles/880886)
- [Sitecore and Microsoft Azure SDK Compatibility Table](https://kb.sitecore.net/articles/026831)
